# 上传 GitHub 说明

本目录为 **LINSY 商城 WEB 端交易链路 Demo** 独立包，可直接上传 GitHub 发布或部署 GitHub Pages。

## 方式一：GitHub 网页上传（最简单）

1. 登录 [GitHub](https://github.com)，点击 **New repository**
2. 仓库名建议：`linsy-mall-web-demo`（Public）
3. 不要勾选 “Add a README”（本包已含 README）
4. 创建后进入仓库，点击 **Add file → Upload files**
5. 将 zip 解压后的**全部文件**拖入（含 `.github` 文件夹）
6. 提交 Commit

### 开启 GitHub Pages

1. 仓库 **Settings → Pages**
2. **Build and deployment → Source** 选 **GitHub Actions**
3. 推送后 Actions 会自动部署；访问地址：

   `https://<你的用户名>.github.io/linsy-mall-web-demo/`

## 方式二：Git 命令行推送

```powershell
cd 商城WEB端交易链路

git init
git add .
git commit -m "Add LINSY mall WEB trade chain demo"

# 先在 GitHub 创建空仓库，再执行（替换 YOUR_USER 和 REPO）
git remote add origin https://github.com/YOUR_USER/linsy-mall-web-demo.git
git branch -M main
git push -u origin main
```

## 方式三：GitHub CLI 一键推送

安装 [GitHub CLI](https://cli.github.com/) 后，在本目录运行：

```powershell
.\push-to-github.ps1
```
